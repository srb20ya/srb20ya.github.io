4.32 BUG REPORT PAGE

DO NOT REPORT MISSING TEXTURES, OR COLORED LIGHTING NOT WORKING - we already know about these,

and they are due to using Demo 4.1's maps with 4.32! (If you are a level designer, you can get these features to work).

 

List of Known Bugs - please do not report ones that are already on this list!

Blue = Fixed in the 4.35 Update (also covers all 4.34 and 4.33 fixes)

Green = Fixed in the 4.34 Update (also covers all 4.33 fixes)

Red = Fixed in the 4.33 Update

 

                * In GFZ1 (and possibly elsewhere), you can jump very high out of shallow water.

                * Knuckles can sometimes "climb on air" below a platform.

                * Lives cannot be lost

                * Race mode does not work

                * CTF mode does not work

                            Workaround for CTF -

                                    The server starts on a gametype other than CTF. All players (including server) start up their game

                                    by using the -ctfteam <1 or 2> parameter (1 is red team, 2 is blue team). After all players have joined,

                                    the server can change the gametype to CTF and it will work.
