Sonic Robo Blast II: Halloween TEST.

Several cheats for this short test version of the game, to use any of these, bring down the console using the tilde (`) key and type in the following:

Cheat What it does...
GOD Invulnerability.
NOCLIP 'Ghost Mode' - allows you to walk through walls.
FLY Fly mode - press and hold the jump button the fly.
EXITLEVEL End the level immediately.
GRAVITY Change the gravity level of the game, minus values will cause Sonic to levitate when you jump. Default gravity is 0.5
TURBO Modifies how fast Sonic can go - maximum is 255%
